# Third-Party SaaS Risk Analytics Dashboard

## Overview
Simulates SaaS vendor risk scoring system used in third-party security assessments.

## Features
- Risk scoring model
- Vendor categorization (Low/Medium/High)
- SQL-based aggregation
- BI-ready dataset export

## Tech Stack
Python, Pandas, Scikit-learn dataset, SQLite, SQL, Risk Analytics

## Output
- Risk-scored dataset
- Category summary table
- Dashboard-ready CSV
